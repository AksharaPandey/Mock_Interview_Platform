# AI Mock Interview Platform 🚀

A premium, AI-powered platform designed to revolutionize interview preparation. By leveraging cutting-edge AI avatars and deep resume analysis, we provide a realistic, low-stakes environment for candidates to refine their skills and boost their confidence.

---

## ✨ Features

- **🎙️ Real-time AI Interviews**: Engage in natural, voice-based conversations with **Sarah**, our advanced AI interviewer, powered by **Vapi AI**.
- **📄 Deep Resume Analysis**: Upload your resume (PDF/Docx) for instant, actionable insights and tailored interview questions generated by **Google Gemini AI**.
- **📊 Performance Dashboard**: Track your progress with a comprehensive interview history, performance metrics, and skill breakdowns.
- **🛠️ Realistic Simulations**: Experience industry-standard interview formats with dynamic questioning based on your specific career path.
- **🎨 Premium UI/UX**: A sleek, dark-themed interface built with **Next.js**, **Tailwind CSS**, and **Radix UI** for a seamless user experience.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)

- **Voice/Avatar Agent**: [Vapi AI](https://vapi.ai/)

- **Real-time Communication**: [Daily.js](https://www.daily.co/)

### Backend & Data
- **Database/Auth**: [Firebase](https://firebase.google.com/)
- **Serverless**: Next.js API Routes

## 🚀 Getting Started

### Prerequisites

- Node.js 20+ installed
- A Firebase project
- A Vapi AI account and API key
- A Google AI (Gemini) API key

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd Mock_Interview_Platform
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file in the root directory and add the following:
   ```env
   # Firebase Admin SDK
   FIREBASE_PROJECT_ID=your_project_id
   FIREBASE_CLIENT_EMAIL=your_client_email
   FIREBASE_PRIVATE_KEY="your_private_key"

   # AI & Voice Providers
   NEXT_PUBLIC_VAPI_WEB_TOKEN=your_vapi_web_token
   GOOGLE_GENERATIVE_AI_API_KEY=your_gemini_api_key
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view the application.
